# vuecms

> vuecms

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# 包含模块有 1、菜单管理 （添加菜单 菜单列表 编辑菜单）
# 包含模块有 2、文章管理 （添加文章 文章列表 编辑文章）
# 包含模块有 3、规则管理 （添加规则 规则列表 编辑规则）
# 包含模块有 4、角色管理 （添加角色 角色列表 编辑角色）
# 包含模块有 5、管理员管理 （添加管理员 管理员列表 编辑管理员）
# 包含模块有 6、登录（header头信息 包含 token 验证）
# 包含模块有 6、规则验证（X）
```
